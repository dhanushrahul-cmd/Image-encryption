# Image Encryption Project

This project provides a simple web interface for encrypting images using the AES (Advanced Encryption Standard) algorithm.

## Features

- Allows users to select an image file from their local system for encryption.
- Encrypts the selected image using AES encryption with a randomly generated key.
- Provides a download link to download the encrypted image.

## Installation

1. Clone this repository to your local machine:

    ```
    git clone https://github.com/your-username/image-encryption.git
    ```

2. Install the required Python dependencies using pip:

    ```
    pip install -r requirements.txt
    ```

## Usage

1. Start the Flask server by running the following command:

    ```
    python app.py
    ```

2. Open your web browser and go to `http://localhost:5000`.

3. Select an image file using the file input field.

4. Click the "Encrypt Image" button to encrypt the selected image.

5. Once the encryption process is complete, a download link will appear. Click on the link to download the encrypted image file.
